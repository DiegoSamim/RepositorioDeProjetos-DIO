Sobre o HTML

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem padrão para a criação e o design de páginas da web. Ele fornece uma estrutura básica para os documentos da web, permitindo a organização e formatação de conteúdo atráves de tags.

Tags

Voltar

Tags HTML são elementos usados para estruturar e apresentar conteúdo em uma página da web. Elas são envolvidas por colchetes angulares e geralmente vêm em pares de abertura e fechamento. A tag de abertura contém o nome da tag, e a tag de fechamento é semelhante, mas inclui uma barra antes do nome

Tag Head

A tag head em HTML é usada para incluir informações sobre o documento que não são exibidas diretamente no navegador. Ela contém meta-informações, como o título da página, links para folhas de estilo (CSS), scripts etc.

Tag Body

A tag em HTML é usada para envolver todo o conteúdo visível de uma página da web. Tudo o que você vê na janela do navegador, como texto, imagens, links, formulários e outros elementos, é colocado dentro da tag body

Tag a

A tag a em HTML é usada para criar links, permitindo a navegação entre diferentes páginas da web ou recursos online

Possui o atributo href no qual será colocado o link da página web. Também pode se tornar uma âncora (levar diretamente para alguma seção da página atual) ou ter outra funcionalidade como baixar algum arquivo etc.

Tags de Texto

Voltar

H1, H2. H3...

As tags h1 a h6 em HTML são utilizadas para definir cabeçalhos ou títulos em uma página web. Elas representam diferentes níveis de hierarquia, sendo h1 o mais importante e h6 o menos

Tag p

A tag p em HTML é usada para definir parágrafos. Ela é fundamental para estruturar os textos em uma página da web.

Tag Blockquote

A tag blockquote em HTML é usada para definir blocos de texto que são citações de outras fontes. Possui uma margem automatica para indentificar que é uma citação.

Exemplo:

"Quando a vida lhe fechar uma porta, apenas abra a porta. É assim que portas funcionam. -Sun Tzu"

Tags de Lista

Voltar

Tag UL

A tag ul em HTML é usada para criar listas não ordenadas. Uma lista não ordenada geralmente consiste em itens, cada um precedido por um marcador, como um ponto, um quadrado ou um círculo.

Tag OL

A tag ol em HTML é usada para criar listas ordenadas, onde cada item é numerado. A estrutura básica de uma lista ordenada usando a tag ol é semelhante à lista não ordenada, mas os itens são precedidos por números em vez de marcadores.
